# PsychoType

PsychoType is a polished static landing page for a typing-psychology experience. The page introduces the product concept, explains the signal layers, and includes a small client-side demo that produces playful writing archetypes from a short text sample.

## Files

- `index.html` — page structure and marketing copy.
- `styles.css` — responsive visual design, layout, and animations/glass effects.
- `script.js` — interactive demo logic for generating a sample PsychoType snapshot.

## Run locally

Open `index.html` in a browser, or serve the folder with a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
